# sass-starter

Special dedication for Compass children. Seriously you do not need Ruby.

## Required

* Node.js + npm

## Installation steps

```bash
$ git clone https://github.com/rootsher/sass-starter.git
$ cd ./sass-starter/
$ sudo npm install -g gulp
$ npm install
```

## How to run?

```bash
$ cd ./sass-starter/
$ gulp
```
